###############################################################################
#                                                                             #
#                       To execute the test for premier:                      #
#                                                                             #
###############################################################################

    Step_1: 
        Make sure the file "master_node" is present in this directory
        This file is generated by the command "deploy.sh" in the "deploy"
        directory. If you want to run the project locally:
            $> echo "localhost" > "master_node"

    Step_2: 
        Launch the worker
            $> celery worker -A work

    Step_3:
        Launch the master
            $> python master.py


###############################################################################
#                                                                             #
#       To execute another makefile (here we provide the one from premier)    #
#                                                                             #
###############################################################################

    Step_4:
        Remove "Makefile" and "premier.c" (optional, but it's cleaner)
            $> rm Makefile premier.c

    Step_5:
        Copy every file from the test directory directly here
            $> cp TEST_DIR/* .

    Step_6:
        And repeat steps 1 to 3


###############################################################################
#                                                                             #
#                                   Notes                                     #
#                                                                             #
###############################################################################

    We use NFS to share files amongst nodes

    This is the main reason why you need to copy files and execute everything
    in the same directory
